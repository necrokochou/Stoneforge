Controls:
Press A key to move left.
Press D key to move right.
Prress Space key to jump. Twice for double jump.
Mouse left button = Attack or Destroy object
E key = Interact with certain objects


Goal:
Mine rock piles to get Gems.
Take down enemies (optional).
Interact with altar to win the game (requires you to have at least 3 gems).


Debug:
F1 key for debug.
